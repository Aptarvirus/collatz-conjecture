# Collatz Conjecture Visualizer
### This is a simple Python implementation of the Collatz Conjecture also know as (3n +1 problem)
## How It works 
## Collatz Conjecture 
### follows two simple rules pf any positive integer n:

### * If n is is even: divide it by 2(n/2)
### * If n is odd : multiply it by 3 and add 1(3n+1)

# Features
## Input Validation:
### Use try-except blokc to handle non integer inputs. preventing the program from crashing
## Error Handling: 
### Checks for numbers less than or equal to $0$ or the number $1$ to ensure valid operation.
